# CV dinámico — GitHub Pages

Este repositorio publica tu **Hoja de Vida** como un sitio web (ideal para compartir en LinkedIn, portafolio y reclutadores).

## Cómo publicarlo (GitHub Pages)
1. Sube estos archivos a tu repositorio (raíz):
   - `index.html`
   - `styles.css`
   - `script.js`
   - carpeta `assets/`

2. Ve a **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: `main`
   - Folder: `/ (root)`

3. Espera a que GitHub muestre la URL pública del sitio.

## Personalización rápida
Edita `index.html`:
- Cambia correo / LinkedIn / ubicación.
- Ajusta el resumen profesional.
- Completa métricas en la sección de experiencia.

## PDF (opcional)
Si quieres botón de "Descargar PDF", agrega tu PDF público dentro de `assets/` con el nombre:
`assets/CV_Katerin_Sanchez.pdf`

> Recomendación: no publiques datos sensibles (documento, dirección exacta, etc.).

## Certificados (descarga)

Los PDFs de certificados están en:

- `assets/certificados/`

Para añadir más, súbelos a esa carpeta y agrega una tarjeta nueva en la sección **Cursos y certificados** dentro de `index.html`.

> Consejo: si tu repo es público, evita publicar documentos con datos sensibles.
